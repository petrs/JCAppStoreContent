How to use the applet

No installation parameters are required. The aplet can be communicated with using OpenSC tool (command line). OpenSC provides various functionality for many various applets, only some of the commands will work. For example, in the description you can see PKCS#15. Then you need to use openSC tool for PKCS#15.

Checking installation

Send opensc-tool --name to see whether OpenSC recognizes the card. You should see 'Javacard with IsoApplet' written in the output.

Applet initialization

Use pkcs15-init -C command and follow the instructions. More info in the personalisation link.

Applet details

Use pkcs15-tool --list-info to get details about the IsoApplet.

Upload your key onto card

Send pkcs15-init -S <your key file> --auth-id <your pin id>. The key should be in PEM or DER format. Your PIN id can be displayed using pkcs15-tool --list-pins - see the ID column.
