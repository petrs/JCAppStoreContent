How to use the applet

Install the applet - no installation parameters required. Once installed, you can use an android app to upload your keepass credentials onto card. The link to the app along with detailed tutorial can be found in the web links section below

Note: In the tutorial, you can skip almost the whole "pyApduTool" section as the store do this for you automatically. Only thing you have to do yourself is to send custom APDU command to the applet once installed - go to your card and select the KeepasNFC applet. Click 'send custom command' and pass command B075000000 to tell it to generate new keypair. Now, the applet has generated the keypair (assuming the card master key is default 4041..4F, e.g. we didn't have to provide it when using gp).

Caveat: This applet is not considered to be ultimately secure (for common usage will suffice though): http://javacardos.com/javacardforum/viewtopic.php?t=998#p3043.
