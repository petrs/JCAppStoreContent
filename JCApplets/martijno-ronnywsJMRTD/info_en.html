How to use the applet

Download the host application and install it. If you have 64bit java and the installer thinks you don't have java, you need to download 32bit java version.

Install the applet onto card. No parameters are required. If the installation fails due to technology support issues, we recommedn to install the no-EAC applet instead.

Use the host application to manage passports, and eventually upload them onto card (open passport in edit mode, select Tools -> Upload Passport).

The uploaded passport should be readable from any ePassport compatible reader.
